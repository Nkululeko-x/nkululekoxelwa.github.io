Android App

Palindromes App

Write an Android Application that will take two numeric input values from the user between 10000 – 99999 and determine if these numbers are palindromes. Should these numbers be palindrome then they are to be added together and the results should be displayed. Finally, the result of the addition should also be checked if it is a palindrome and displayed to the user. Please validate user input, use appropriate controls for input, output.
